## All tasks

### `attachment`

Gets an attachment. [Official Linear docs](https://developers.linear.app/docs/graphql/attachments).

```ts example.ts
const attachment = await io.linear.attachment("get-attachment", { 
  id: "<attachment-id>" // the id of the attachment
  });
```
